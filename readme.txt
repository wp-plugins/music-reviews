== Description ==

Custom Music review plugin support you to add an new post type with taxonomy, custom fields, feature images. Here you can add this into your wordpress site easily.

== Installation ==

1. Upload the entire `custom-music-reviews` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Custom Music Reviews' menu in your WordPress admin panel.
